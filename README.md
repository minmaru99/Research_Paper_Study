## Team Rules

1. **질문 및 답변 정리**
   - 질문자는 **질문 내용**과 **답변 내용**을 정리합니다.
   - 각자 **소감**을 작성한 뒤, 끝난 후 이를 취합하여 하나의 파일로 정리합니다.

2. **파일 관리**
   - GitHub 리포지토리에서 아래의 파일을 체계적으로 관리합니다:
     - 논문 원본 및 번역본
     - 발표 자료
     - 질의응답 내용
     - 각자의 소감

3. **질문 진행 방식**
   - 질문은 발표가 끝난 후에 진행합니다.
   - 답변이 잘 되지 않은 질문은 **다음 주 발표 때까지 준비**하여 다시 답변합니다.
